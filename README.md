# Parabola Art Maker

## Overview
Parabola Art Maker is an interactive application that allows users to create stunning graphical art based on parabolic curves. The application leverages advanced mathematical algorithms to generate beautiful designs that can be customized and exported.

## Features
- **Interactive Design:** Draw and manipulate parabolic curves.
- **Customizable Colors:** Change the color scheme of your designs.
- **Export Options:** Save your creations in various formats (e.g., PNG, SVG).
- **Undo/Redo Functionality:** Easily revert changes or redo actions.
- **Zoom and Pan:** Navigate through your artwork with ease.

## Usage Instructions
1. **Installation:** Download and install the application from [link to download].  
2. **Starting the Application:** Launch the application from your applications menu.
3. **Creating Art:** Use the drawing tools to create your designs.  
4. **Exporting Your Work:** Click on the 'Export' button and choose your preferred file format.

## Tips and Tricks
- Experiment with different curve parameters to achieve unique designs.
- Use the preview function to see how changes affect the overall composition before finalizing.
- Regularly use the save function to avoid losing your work.

## Keyboard Shortcuts
- **Ctrl + N:** Create a new file.
- **Ctrl + S:** Save the current artwork.
- **Ctrl + Z:** Undo the last action.
- **Ctrl + Shift + Z:** Redo the last undone action.
- **Ctrl + E:** Export the artwork.

## Technical Details
- **Programming Language:** The application is built using Python.
- **Framework Used:** Pygame for rendering graphics.
- **Math Libraries:** NumPy for numerical calculations.
